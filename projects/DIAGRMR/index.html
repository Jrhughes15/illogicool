DIAGRMR Help

Basics

Right-click boxes to add plot points, branches, characters, notes, scene/chapter boxes, time boxes, meta boxes, and to change plot type, color, or border style.

Type directly in newly created boxes. Press Enter to finish. For new branches and rails, Tab moves from the new location box to the new plot box.

Right-click empty grid space to add rails, insert or remove horizontal timeline space, toggle panels, or switch modes.

Auto Shift opens timeline space when plot boxes are added or widened. Turn it off when you want simultaneous/manual diagramming.

Moving And Selection

Drag plot and location boxes from the left or right edge area. The center of a box is reserved for text editing and selection.

Ctrl-click or Cmd-click boxes to select multiple boxes, then drag them together or use the selection context menu.

Navigation

Drag the canvas background to pan. Hold the middle mouse button and move the mouse to auto-pan from that point.

Topmost, Bottommost, Leftmost, and Rightmost jump to the diagram extremes. Fit Height fits the top and bottom of the current diagram in view.

Zoom buttons and wheel zoom anchor to the selected box when one is selected; otherwise they use the viewport center or mouse position.

The map can be clicked or dragged to pan the viewport. Use its Map tab or the grid context menu to hide or show it.

Lines And Boxes

Use Line Mode to connect edge-center handles between boxes. Double-click a connector to enter line mode. Press Esc to leave line mode or cancel a pending connector.

Right-click a connector handle or connector line to delete it, change its line type, reset its route, change ports, or add arrows.

Use Box Mode to draw encompassing boxes with editable note labels. Right-click the box or note to change line type, note placement, or delete it.

Box Types

Location boxes are yellow rail anchors. Plot boxes hold story events and can be Beginning, End, Normal, Dialogue, Death, or Speculation.

Character boxes are blue attachments below plot boxes. Larger character stacks wrap into columns of four. Note boxes are small white annotations that attach beside or below their parent box.

Film/Meta, Scene/Chapter, and Year/Time markers sit above the plot rails. Film and scene/chapter markers also add a yellow span above the plot.

Time Passage boxes are vertical grey markers spanning location rails. Box Mode boxes are dashed enclosing regions with a small editable note label.

Location Diagram

The Locations side tab shows a condensed organization column for groups, subgroups, sub-locations, first/last plot jumps, character presence, and marker rails.

Right-click location boxes there to organize groups, subgroups, and sub-locations without changing the main diagram layout. The Manager includes a Location Org tab for the same hierarchy.

Marker rows appear above location rails in this order: Films, Scenes/Chapters, then Years. Each marker appears as its own clickable segment. Film and scene/chapter markers also create a yellow span directly above the related plot box; deleting either paired marker removes both.

Gather and move group rails from location context menus when you need to reorganize a cluster of related locations.

In the location diagram, the left half of a blank plot box jumps to the first plot on that rail; the right half jumps to the last plot.

View And Export

Grid toggles the viewport grid. Dark switches between light and dark themes. The canvas context menu includes common view, panel, and mode toggles.

The Manager includes Info, Styles, detailed Stats, Export, Location Org, Help, and searchable lists for locations, plots, notes, characters, markers, groups, and story details. Info stores title, author, dates, source/status, summary, and overall notes. Styles lets you click swatches to choose safe pastel colors, add custom plot colors, and label line types for this diagram.

Export creates an SVG of the used diagram bounds with a margin. You can include the location diagram, an optional used-styles legend, and choose grid, white, or transparent background.

Use the Manager to search and jump through locations, plots, characters, connectors, groups, story notes, tags, and the location organization tree.

The Details drawer shows selected-item specs and stores longer notes and thread/theme tags for the selected box, connector, or enclosing box. Search opens from the magnifying glass, highlights matches, supports scope toggles, and lets you jump through results with previous/next controls.

L

B

Middle-hold
Auto-pan from the held point until the wheel button is released.
Ctrl/Cmd+Enter
Add a plot point after the selected box.
Ctrl/Cmd+Shift+C
Add a character below the selected plot or character box.
Ctrl/Cmd+Shift+B
Add a new branch below the selected plot box.
Ctrl/Cmd+F
Open DIAGRMR search when not editing text.
Esc
Close menus, leave line mode, or cancel connector work.
