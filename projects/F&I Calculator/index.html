Help

How entry works

  • Pressing an operator (+ − × ÷ ) or Evaluate commits what is in Entry first.
  • Default is length with feet implied.
  • To force a number, end it with # (example: 8#).

Cheat sheet

  • 4 → 4'
  • 4'10 → 4'10"
  • 10" → 10"
  • 8# → number 8
  • 2.5# → number 2.5

Rules

  • length + length, length - length
  • number × length, length × number
  • length ÷ number
  • No length × length (that becomes area)
